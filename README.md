# React Native Demo App

This is a simple demo React Native application built with Expo that shows how to use Stark's accessibility scanner.

To test sending results to your [Stark dashboard](https://app.getstark.co), follow the instructions at [https://www.getstark.co/support/getting-started/using-source-code-integrations-for-mobile-frameworks/](https://www.getstark.co/support/getting-started/using-source-code-integrations-for-mobile-frameworks/)


## Getting Started

```bash
# Install dependencies
npm install

# Start Expo development server
npm start

# Run on iOS
npm run ios

# Run on Android
npm run android

# Run on browser
npm run web
```
